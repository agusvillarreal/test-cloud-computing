Welcome to Docker!

This is a simple web server running inside a Docker container using Nginx.

Container Information:

  • Base Image: nginx:alpine
  • Port: 80
  • Web Server: Nginx
  • Status: Running in Docker container

How to run this example:

# Build the image
docker build -t my-web-server .

# Run the container
docker run -p 8080:80 my-web-server

# Access the website
# Open http://localhost:8080 in your browser

Docker Concepts Demonstrated:

  • Using official base images (nginx:alpine)
  • Copying files into the container
  • Port exposure and mapping
  • Running web servers in containers
